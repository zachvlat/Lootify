# Lootify

Discover and claim free games from your favorite game stores!

Lootify helps you keep track of free games available on Steam, GOG, Epic Games Store, and other major platforms. Never miss out on a free game again!

## Features

- 🎮 Browse free games from multiple stores
- 🎯 Curated collection of available claims
- 📱 Android mobile application

## About

This is an Android application built with Kotlin that aggregates free game offerings from popular game distribution platforms.
